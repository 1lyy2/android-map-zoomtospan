# android-map-zoomtospan
地图内容全览

本工程为基于高德地图Android SDK进行封装，实现通过缩放保证地图上自定义内容（模拟为点）均展现在视野范围内, 并且地图中心点不被改变。。
## 前述 ##
- [高德官网申请Key](http://lbs.amap.com/dev/#/).
- 阅读[参考手册](http://a.amap.com/lbs/static/unzip/Android_Map_Doc/index.html).
- 工程基于3D地图SDK实现
